# VibeScribe 🎙️

VibeScribe is a premium, high-fidelity audio transcription engine designed for the modern era. Leveraging state-of-the-art AI, it transforms your audio data into crisp, actionable intelligence with a focus on sophisticated design and surgical clarity.

## ✨ Features

- **Premium Glassmorphism UI**: A surgically clean, modern SaaS interface featuring frosted glass aesthetics and a light-theme philosophy.
- **Intelligent Transcription**: High-fidelity extraction of data from various audio formats.
- **Data Stream Visualization**: Watch your audio metadata transform into structured intelligence.
- **Secure Access Matrix**: Seamless and secure authentication system.
- **Transcript Management**: Organized, searchable list of your recent intelligence captures.
- **One-Click Synchronization**: Instantly clone transcription text to your clipboard.

## 🎨 Design System

VibeScribe follows a strict, high-contrast palette:

- **Moss Green (#8F9D68)**: Grounded, professional sidebar and action tones.
- **Old Lace (#FFF8EB)**: Clean, high-visibility background surface.
- **Iris Purple (#7B57CE)**: Vibrant accent and specialized glass transcription cards.

## 🛠️ Technology Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS with custom Glassmorphism Primitives
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Authentication**: Better Auth
- **Database/ORM**: Drizzle ORM + PostgreSQL
